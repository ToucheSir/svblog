svblog
=========

A basic blog application written using Python, Flask, Jinja2, and HTML, allowing users to make blog posts and upload files.

Use the following directory structure when deploying:

    svblog/
        data/
            (generated .db files)
        uploads/
            (image files)
        static/
            style.css
        templates/
            index.html
            create.html
            login.html
            entries.html
            upload.html
        app.py
        tools.py
        user.py
        upload.py
        entry.py
        settings.cfg
